## 🎧 About
Phonk-Hub is a simple web-based music hub made for exploring and listening to phonk-style music. It’s built to be fast, lightweight, and easy to use.

## 🚀 Features
- Clean and minimal design  
- Fast loading site  
- Mobile-friendly layout  
- Phonk music focus
